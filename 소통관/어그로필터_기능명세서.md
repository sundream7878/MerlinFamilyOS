# 📝 어그로필터 기능 명세서
- **작성 주체**: **개별앱 AI (윈드서퍼)** 가 본문의 상세 내용을 채우고 상시 최신화한다.
- **활용 용도**: 허브 AI가 앱의 로직을 이해하고 연동(지갑, 인증 등) 설계를 할 때 참조하는 **최종 권위 문서**.
- **작성 가이드**: 
  1. 앱의 고유 비즈니스 로직(분석, 보상, 알림 등)을 상세히 기술할 것.
  2. 허브와 연동되는 지점(결제 금액, 유저 식별 방식)은 반드시 명시할 것.
  3. 허브 AI는 이 문서를 바탕으로 연동 코드를 작성하므로, 변경 시 즉시 보고 및 갱신 필수.

---

## 1. 서비스 개요
- **목표**: 유튜브 영상의 신뢰도 분석 및 어그로 필터링
- **주요 유저**: 유튜브 시청자 및 분석가

## 2. 핵심 기능 리스트
- **[분석 엔진]**: 
  - 사용자가 입력한 유튜브 URL에서 영상 ID 및 자막(Transcript), 메타데이터(조회수, 게시일 등) 추출
  - 자체 알고리즘을 통해 낚시성(Clickbait), 정보의 정확성(Accuracy), 종합 신뢰도(Reliability) 점수 산출
  - 분석 결과 요약 및 AI 권장 제목 제공
- **[코인 시스템 (허브 중앙 관리)]**: 
  - 신규 영상 분석 요청 시 허브 API를 통해 코인 차감 (단가는 `family_aggro_video_pricing`에서 확정)
  - 이미 분석된 영상 재조회 시에는 차감 없이 즉시 노출 (캐시 히트)
  - 코인 잔액/입출금은 허브 `family_coin_balances` / `family_coin_history`에서 일괄 관리
- **[구독 및 랭킹 알림]**: 
  - 관심 유튜브 채널 구독 시 해당 채널의 신규 분석 영상 자동 추적
  - 채널의 종합 신뢰도 등급 변화(예: B -> A) 발생 시 유저에게 실시간 알림 발송
  - 카테고리별 채널 랭킹 산출 및 상위 10% 진입 시 축하 알림 발송

## 3. 연동 필요 접점
- **유저 식별**: 허브 UUID (`family_users.id`) 단일 사용. localStorage 키: `merlin_user_id`
- **인증**: 허브 OTP/JWT 연동. (내부적으로 **Supabase Auth** 및 **@supabase/ssr** 사용)
- **코인**: 허브 `/api/wallet/transaction` API를 통한 코인 차감/조회. (내부적으로 **pg(node-postgres)** 커넥션 풀 사용)
- **이미지**: 유저 프로필 이미지 표시. (Hub SSOT 원칙에 따른 200px Base64 데이터 교환)
- **핵심 엔진**:
  - **YouTube Data API v3**: 영상/채널 메타데이터 추출 의존성
  - **Google Gemini API (@google/genai)**: 2.5-flash 모델 기반 신뢰도 분석 의존성

## 4. 특이사항 및 제약조건
- **DB 라이브러리**: **Supabase-js**를 기본으로 하되, 복잡한 트랜잭션 처리를 위해 **pg (node-postgres)** 라이브러리를 병행 사용함.
- **유튜브 API 쿼터**: 대량 분석 요청 시 Google API Quota 소진 대비 캐싱 전략 운영.
- **데이터 보존**: 분석된 영상 데이터 및 통계 자료의 영속적 보관 로직 포함.
- **환불 규정**: 디지털 재화 특성상 7일 이내 및 미사용 시에 한해 청약 철회 가능.
